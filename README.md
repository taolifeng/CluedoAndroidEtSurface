Cluedo
======

Pour windows 7 : problème d'ip dû au réseau virtual host only
Diagnostiquer le problème:
Ouvrir un terminal: ipconfig
Si vous avez un réseau ethernet virtual host only cela va vous poser problème
Il faut aller dans le centre réseau et partage et désactiver ce réseau
Le problème est alors résolu c'est le bon ip qui est généré à la volée

Récupération du code

­ Télécharger l’archive du code des différents supports
­ Extraire le contenu du dossier


Installation Téléphone Android

Première possibilité :

­ Aller dans CluedoPhoneAndroid > app
­ Récupérer cluedo­perron.apk
­ Le copier/télécharger sur un téléphone android
­ Installer le fichier


Deuxième possibilité :

­ Disposer du SDK Android et de Android Studio
­ File > Open > path/CluedoPhoneAndroid
­ Brancher un téléphone android
­ Run (entraine l’installation du fichier sur le téléphone)
­ Choose a running device > OK


Installation Tablette Android

­ Disposer du SDK Android et de Android Studio
­ File > Open > path/CluedoTabAndroid
­ Brancher un téléphone android
­ Run (entraine l’installation du fichier sur le téléphone)
­ Choose a running device > OK


Installation Table Surface

­ Disposer de Visual Studio
­ Copier le dossier “CluedoSurface” sur la table surface
­ Aller dans CluedoSurface
­ Double clique sur Cluedo.sln avec ouverture par Visual Studio
­ Run (lancement de l’application sur la table)


Premières étapes

­ Lancer l’application sur la table
­ Lancer l’application sur les téléphones et tablettes
­ Scanner chacun un qr_code
­ Lancer la partie (bouton play)
