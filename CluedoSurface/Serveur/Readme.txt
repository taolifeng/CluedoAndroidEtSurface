# Installation de NodeJS (Sous Windows)
- http://nodejs.org/download/
- Télécharger l'installeur correspondant à votre système.
- Ouvrir un prompt "cmd"
- Lancer la commande npm : une erreur devrait survenir...
- Ajouter un répertoire "npm" à l'endroit où se situe le problème
- Ajouter le dossier d'installation de NodeJS à la variable d'environnement PATH (C:\Program Files\nodejs)

# Installation de PaintServer
- npm install
- tester le bon fonctionne du serveur en exécutant startServer.bat
- Autoriser l'accès au réseau
